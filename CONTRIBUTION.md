# Contribution Guide

Thank you for considering contributing to this project!

## Getting Started

1. Fork the repository and clone it locally.
2. Make sure you have `pnpm` installed.
3. Run `pnpm install` to set up the project dependencies.

## Development

- Use TypeScript for all code contributions.
- Ensure code style consistency with the existing codebase.
- Run `pnpm test` to ensure all tests pass before submitting a pull request.

## Pull Requests

- Describe your changes clearly and why they are needed.
- Reference any relevant issues in your pull request.
- Ensure all new and existing tests pass.

We appreciate your contributions and thank you for your time!
